# e-plantShopping

Welcome to **e-plantShopping**, a front-end web application project built as part of the *Developing Front-End Apps with React* course.

## Project Overview

e-plantShopping is an online plant shopping platform where users can browse, search, and purchase a variety of plants. This project helps you practice building interactive React applications with modern front-end development tools.

## Features

- Browse a catalog of plants with images and descriptions
- Search and filter plants by category or name
- Add plants to a shopping cart
- Responsive design for desktop and mobile devices
- State management using React hooks
- Navigation with React Router

## Development Environment

This project is developed using the Cloud IDE tools provided in the course environment:

- Base working directory: `/home/project`
- React framework with Vite (preferred)
- Development and testing directly in the Cloud IDE
